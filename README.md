Sistema de eventos – TProg (Facultad de Ingeniería)

Proyecto desarrollado en el marco de la materia Taller de Programación (TProg) del segundo semestre de tercer año de la carrera de Ingeniería en Computación.

Descripción

Sistema de gestión de eventos implementado en Java, con arquitectura distribuida basada en servicios web (SOAP) y aplicación web para interacción con usuarios.

El sistema permite el registro, consulta y gestión de usuarios y eventos, incluyendo funcionalidades de seguimiento y participación.

Características

Aplicación web basada en JSP/Servlets desplegada en Apache Tomcat
Comunicación mediante servicios web (SOAP)
Separación entre cliente y servidor
Persistencia de datos mediante base de datos (opcional implementada)
Generación de stubs a partir de WSDL


Tecnologías utilizadas
Java
Jakarta EE (Servlets, JSP)
Apache Tomcat
JAX-WS (SOAP)
Maven


Notas

Se implementaron todas las funcionalidades opcionales propuestas en la consigna, incluyendo persistencia en base de datos y carga de imágenes.
